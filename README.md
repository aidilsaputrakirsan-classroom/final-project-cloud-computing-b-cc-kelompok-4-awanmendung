Resepin.id adalah website berbasis Laravel yang menyediakan kumpulan resep masakan Indonesia maupun internasional.
Platform ini dirancang agar pengguna dapat:

Menemukan resep masakan berdasarkan kategori atau popularitas.

Memberi rating pada resep berdasarkan pengalaman memasak.

Menambahkan komentar dan berbagi feedback.

Menyimpan resep favorit ke menu Bookmark agar mudah diakses kembali.

Melihat informasi lengkap resep: bahan, langkah memasak, waktu memasak, tingkat kesulitan, dan nilai gizi (opsional).
